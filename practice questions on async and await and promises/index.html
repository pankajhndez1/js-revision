async and await practice problems !!

Ques : Write a function that takes in an array of URLs and fetches the contents of each URL using fetch and async/await. The function should return an array of the fetched contents.


Ques : Write a function that takes in two functions that return Promises and runs them in parallel using Promise.all. The function should return the combined results of the two Promises.


Ques : Write a function that takes in a number and returns a Promise that resolves after the number of milliseconds specified.


Ques : Write a function that takes in a string and returns a Promise that resolves with the length of the string. If the string is empty, the Promise should reject with an error message.
